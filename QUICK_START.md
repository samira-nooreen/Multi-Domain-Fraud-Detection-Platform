## If you need to restart:

### Option 1: Use the batch file (Recommended)
1. Double-click `start_app.bat`
2. Wait for "Running on http://127.0.0.1:5000"
3. Open browser to http://127.0.0.1:5000

### Option 2: Manual start
```powershell
# Close ALL existing Python/Flask processes first
# Then run:
.venv\Scripts\python.exe app.py
```

---

## ⚠️ Important Notes:

1. **Don't use `python app.py`** - use `.venv\Scripts\python.exe app.py` instead
2. **Check if app is already running** before starting a new instance
3. **The virtual environment has numpy/pandas issues** - we're working on fixing it
