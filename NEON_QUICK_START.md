## 🎨 Quick Start

### See It In Action
1. Open browser: `http://localhost:5000/neon-example`
2. You'll see a stunning hero section with neon rings (like your metaverse image!)
3. Scroll to see feature cards, animated borders, and glowing stats

### Use It On Your Pages
Just add classes to any HTML element:

```html
<!-- Pink button -->
<button class="btn-neon-pink">Click Me!</button>

<!-- Glowing text -->
<h1 class="neon-text-cyan">Hello World</h1>

<!-- Neon card -->
<div class="card-neon">Content here</div>

<!-- Badge -->
<span class="badge-neon-purple">NEW</span>
```

## 💡 Tips

- **Neon rings** (like your metaverse image): Use `.neon-ring-pink`, `.neon-ring-cyan`, or `.neon-ring-purple`
- **Hero sections**: Use `background: var(--gradient-metaverse)` for multi-color gradients
- **CTAs**: `btn-neon-pink` for hot/urgent actions, `btn-neon-cyan` for cool/tech features
- **Cards**: Hover over `.card-neon` elements to see animated border reveal
- **Text**: Use `.neon-text-*` classes for glowing headings
