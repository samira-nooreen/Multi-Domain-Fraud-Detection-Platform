Quick Start Guide

1. Include the CSS files in your HTML:

<link rel="stylesheet" href="static/css/colors.css"> <link rel="stylesheet" href="static/css/neon-effects.css">

2. Use the pre-made classes:

<!-- Buttons --> <button class="btn-neon-pink">Pink Button</button> <button class="btn-neon-cyan">Cyan Button</button> <!-- Text --> <h1 class="neon-text-purple">Glowing Text</h1> <!-- Cards --> <div class="card-neon">Content</div> <div class="glass-neon-pink">Content</div> <!-- Badges --> <span class="badge-neon-cyan">Badge</span>

3. Or use CSS variables directly:

.my-element { background: var(--gradient-metaverse); color: var(--neon-pink); box-shadow: var(--shadow-neon-cyan); }
